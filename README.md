# Vite + React + Tailwind + TypeScript

Quick start:

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

3. Build

```bash
npm run build
```

Notes:
- This project uses Vite, React 18, TypeScript, and Tailwind CSS.
- If you want JavaScript instead of TypeScript, I can convert the files.
